# JZ Portfolio

_Pagina web donde se muestra el Portfolio de JZ_

## Comenzando 🚀

_Una vez descargado ejecutar:_
```
npm install
```

_Instalara  **SASS** y para modificar el archivo scss se debe ejecutar en una terminal:_
```
npm run watch-css
```
